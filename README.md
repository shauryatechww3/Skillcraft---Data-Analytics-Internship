# Skillcraft Data Analytics Internship

This repository contains all four tasks successfully completed during my Data Analytics Internship at Skillcraft Technology.

---

## 📁 Project Structure & Tasks Overview

### 📊 Task 1: Titanic Dataset Visualizations
- **File:** `Skillcraft Task 1 .xlsx`
- **Description:** Created structured data insights and a detailed visual analysis of population demographics and gender distribution using Google Sheets.

### 🐍 Task 2: Data Cleaning & Preparation
- **File:** `Task2_Data_Cleaning_and_Preparation.ipynb`
- **Description:** Loaded the Titanic dataset in Python via Google Colab. Cleaned the data by handling missing values (imputed 'Age' using median), dropping duplicates, and performing string-to-date type conversions.

### 📉 Task 3: Interactive Marketing Dashboard
- **File:** `Task 3 Power BI Dashboard.png`
- **Description:** Built a professional Power BI dashboard using the Bank Product Purchase dataset. Developed interactive visuals including KPI cards for total headcount, a donut chart for marital status splits, and age-based slicers.

### 🚗 Task 4: Business Insights Report (Accident Data EDA)
- **File:** `Task4_Accident_Data_EDA.ipynb`
- **Description:** Performed full Exploratory Data Analysis (EDA) on traffic accident data. Generated professional visualizations aggregating accidents by Weather and Road conditions using Pandas, Matplotlib, and Seaborn.
